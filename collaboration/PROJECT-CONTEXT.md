# Project Context

## Project

A local knowledge-map and future always-on display system.

## Big vision

Build a local AI workspace that:

- understands personal files
- visualizes relationships
- can run on an always-on display machine
- later supports voice updates and remote control

## Current product direction

Not a heavy manual note-taking system.

Instead:

- file-aware visual map
- cluster-first overview
- future AI assistant layer on top

## Current display direction

The user may use:

- main PC for editing and rebuilding
- old MSI machine as display machine
- later maybe iPad as control surface
