# AI Collaboration

This folder is the shared handoff layer between:

- Codex
- Claude
- you

## Goal

Keep one simple source of truth so both AI tools can work on the same project without losing context.

## Main files

- `PROJECT-CONTEXT.md`
- `CURRENT-STATE.md`
- `NEXT-TASKS.md`
- `CLAUDE-HANDOFF.md`
- `DECISIONS.md`

## How to use it

1. ask one AI to read these files first
2. let it do focused work
3. have it update the handoff files
4. switch to the other AI when useful
