# Decisions

## Decision 1

The system should be cluster-first, not graph-everything-first.

## Decision 2

The old MSI machine should be treated as a display machine, not the main compute machine.

## Decision 3

The main PC should do the heavier rebuild and update work.

## Decision 4

The project should remain easy to evolve later toward:

- voice updates
- remote updates
- iPad control
- richer AI behavior
